# Project Zeus - Market Research Automation

Internal Python tool designed for high-performance market research using the Google Ads API.

## Features:
- **Service Used**: KeywordPlanIdeaService (GenerateKeywordIdeas).
- **Functionality**: Analyzes keyword volume and CPC metrics for international markets.
- **Safety**: Read-only operations with no campaign management capabilities.
- **Environment**: Secure local execution (CMD-based).

## Compliance:
This tool is for internal use only and fully complies with Google Ads API Terms and Conditions.
